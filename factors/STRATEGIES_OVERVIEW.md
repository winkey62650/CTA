## 🎯 策略使用说明

### 时间周期推荐
- **1H (1小时)**: 适合短线交易，参数n: 5-20
- **4H (4小时)**: 适合中线交易，参数n: 10-40
- **12H (12小时)**: 适合波段交易，参数n: 20-60
- **24H (日线)**: 适合趋势交易，参数n: 30-200

### 参数n范围说明
- **n_min**: 最小周期，快速信号但噪声多
- **n_max**: 最大周期，稳定但延迟
- **推荐值**: 根据时间周期选择中间值

### 代码使用方法
每个策略文件都包含：
- `signal()` 函数：生成交易信号
- `para_list()` 函数：定义参数范围
- 支持 OHLCV 数据格式

## 📈 使用指南

### 快速开始

1. **选择策略**: 根据市场状态选择策略类型
   - 趋势市 → 趋势跟踪类
   - 震荡市 → 均值回归类
   - 爆发行情 → 突破类

2. **配置参数**: 根据 `factors/` 目录下的策略文件配置参数
   ```python
   # config.py 示例
   symbol_list = ['BTC-USDT']
   para = [20]  # 策略参数
   signal_name_list = ['bollinger']  # 策略名称
   rule_type_list = ['4H']  # 时间周期
   ```

3. **运行回测**:
   ```bash
   # 单次回测
   python 2_fast_backview.py

   # 参数优化
   python 3_fastover.py

   # 评估报告
   python 4_strategy_evaluate.py
   ```

### 参数优化建议

1. **参数高原**: 选择参数值变化对收益影响小的区域
2. **避免过拟合**: 参数范围不宜过窄
3. **样本外验证**: 用历史数据训练，用未来数据验证
4. **鲁棒性测试**: 不同市场环境下参数稳定性

### 组合策略建议

1. **趋势+反转**: 趋势类(60%) + 反转类(40%)
2. **多周期组合**: 4H(30%) + 12H(40%) + 24H(30%)
3. **多品种分散**: 不同相关性品种组合

## ⚠️ 风险提示

1. **回测偏差**: 历史表现不等于未来收益
2. **滑点成本**: 实盘会有额外的滑点和手续费
3. **市场变化**: 策略参数需要定期重新优化
4. **黑天鹅事件**: 极端行情下策略可能失效
